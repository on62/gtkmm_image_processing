# gtkmm_image_processing
## image processing based on gtkmm, opencv

1. Compilation
  * Debug mode compilation
    ```bash
    $ ./build eng
    ```
  * Release mode compilation
    ```bash
    $ ./build
     ```
     or
     ```bash
     $ ./build all
     ```
2. Clean
    ```bash
    $ ./build clean
    ```

3. Execution  
    ```bash
   $ ./magnolia
   ```
